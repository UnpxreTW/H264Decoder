# RawH264DataConverter

將 Raw H.264 Data 轉換為 CMSampleBuffer 或 CVPixelBuffer

## 開發

建議使用 [Tuist](http://192.168.1.95/documentation/home/tuist) 產生的專案檔進行開發

> Note: 使用產生 Xcode 專案開發以包含 `SwiftLint` 與 `SwiftFormater` 等工具

使用指令產生專案檔以進行開發：

```shell
tuist generate
```


> Note: 產生 Tuist 配置檔
>
> ```shell
> tuist edit
> ```
